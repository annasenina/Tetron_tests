# Автотесты на проект https://www.tetron.ru/

## Используемые технологии:
**Тесты:** Java, Gradle, Junit5, Selenide

**Отчетность:** Allure Reports

**CI:** Jenkins

**TMS:** Allure TestOps

**Запуск тестов:** Selenoid
 
**Уведомление:** Telegram Bot

**Task Tracker:** Jira
